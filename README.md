# 🌱 Smart Irrigation System 

A smart irrigation system developed using Arduino to optimize water usage and automate irrigation without internet connectivity.

The system monitors soil moisture and automatically controls water supply, reducing water wastage and improving irrigation efficiency.

---

# 📌 Project Objective

Traditional irrigation methods often lead to unnecessary water usage and manual effort.

This project aims to:

✔ Reduce water wastage  
✔ Automate irrigation process  
✔ Improve irrigation efficiency  
✔ Develop a low-cost agriculture solution  

---

# 🚀 Features

## 💧 Automatic Water Control
Automatically turns the water pump ON/OFF based on soil moisture.

## 🌱 Soil Moisture Monitoring
Continuously monitors moisture level in soil.

## ⚡ Efficient Water Usage
Supplies water only when required.

## 🔄 Fully Automated Operation
Minimizes manual irrigation effort.

## 💰 Low Cost Implementation
Works without internet or cloud services.
